# analyze-skill-design

`analyze-skill-design` 是一个用于拆解和评价 Codex/AI Skill 的分析型 Skill。它从产品视角审视一个 Skill 的触发设计、工作流程、资源组织、脚本自动化、输出质量和可维护性，帮助使用者理解一个 Skill 不只是“能做什么”，更重要的是“为什么这样设计”。

该 Skill 适合用于分析已有 Skill 的优缺点、提炼设计模式、发现改进空间，并生成结构化的 `analysis/howSkills.md` 分析文档。它内置目录扫描脚本、评分 rubric、输出模板、好坏样例和自检清单，让 Agent 能更稳定地产出有证据、有判断、有改进建议的 Skill 设计分析。
